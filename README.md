This project analyzes the relationship between mental health and social media usage using unsupervised machine learning (K-Means clustering). The goal was to identify hidden patterns and group users based on behavioral and lifestyle factors.

Features: 
Data cleaning and preprocessing of real-world datasets
Conversion of mixed data into fully quantitative format
Application of K-Means clustering (Scikit-learn)
3D data visualization to explore patterns and clusters
Interpretation of cluster results and trends
Non-technical conclusions for real-world understanding

Technologies Used
Python
Pandas
NumPy
Scikit-learn
Matplotlib
